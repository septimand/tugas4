Fakultas Teknologi Informasi

Fakultas Teknologi Informasi UKDW sejak awal berdirinya terus berkomitmen untuk menyelenggarakan pendidikan bidang Informatika dan Sistem Informasi yang berkualitas dan berintegritas sesuai karakter Kedutawacanaan.

Program Study

Informatika

salah satu program studi di bawah naungan FTI, Universitas Kristen Duta Wacana. Secara Khusus, Prodi Informatika UKDW menyiapkan lulusannya untuk menjadi seorang System Programmer, Network Service Supervicer, Database Administrator, Software Enginer, User Interface Designer, dan Intelligent System Developher. Lulusan Program Studi Informatika dapat bekerja di bidang teknologi informasi sebagai pengembang aplikasi (baik aplikasi desktop, web dan mobile), pengelola dan supervisi infrastruktur jaringan, pengelola basis data, pengembang aplikasi yang bekerja sesuai dengan metodologi pengembangan perangkat lunak, perancang antar muka program yang mudah dipelajari dan digunakan serta sebagai pengembang suatu sistem cerdas. Kerja sama dengan berbagai mitra pun dilakukan untuk mendukung dan mengembangkan minat akademik maupun non akademik, seperti pihak industri sebagai pengguna lulusan dan Asosiasi Perguruan Tinggi Ilmu Komputer Indonesia (APTIKOM).


Sistem Informasi

program studi yang mempelajari cara meningkatkan kemampuan organisasi dengan menerapkan teknologi informasi ke dalamnya. Informasi berdiri di 2 kaki, yaitu kaki organisasi dan kaki perangkat lunak. Pada kaki organisasi, jurusan system informasi belajar tentang cara mendefinisikan prosedur organisasi, manajemen, dll. Di kaki perangkat lunak, jurursan sistem informasi belajar cara mengimplementasikan basis data dan mengembangkan unjuk kerja organisasi, serta bagaiman dampak perubahan organisasi sebagai akibat implementasi perangkat lunak tersebut. Ditinjau dari jurusannya, jurusan sistem informasi merupakan titik tengah antara Informatika dan Manajemen. Sistem Informasi lebih banyak mempelajari bidang pengetahuan organisasi (baik konsep maupun teknisnya), aplikasi teknologi dan perangkat lunak. Sistem Informasi hanya sedikit membahas tentang infrastruktur sistem maupun perangkat keras komputer.
